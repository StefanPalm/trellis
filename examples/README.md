# Examples

Two things live here:

- **[compounded-learning/](compounded-learning/)** — a small, complete wiki
  built and maintained with the four trellis skills, about the practice of
  compounded learning itself. Browse it to see the destination: a filled
  AGENTS.md, seven interlinked concept pages with provenance and statuses
  (including one honest `contested` page), an archive with intake records,
  a newest-first log, and a lint report. Everything in it is exactly what the
  skills produce.

- **[starter-inbox/](starter-inbox/)** — three raw notes you can copy into
  your own first wiki's `inbox/` to run your first ingest without needing
  material of your own. They are deliberately messy; that's what an inbox
  looks like.

To try the full loop in ten minutes: run `wiki-init` for a wiki about
"learning how to learn with agents", copy the starter-inbox files into its
inbox, run `wiki-ingest`, then `wiki-lint`, then ask it a question with
`wiki-ask`.
